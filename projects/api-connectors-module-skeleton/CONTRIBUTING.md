# Contributing to API Connectors Module Skeleton

## Development Setup
1. Clone the repository.
2. Install project dependencies for the selected module stack.
3. Review `RULES.md`, `PLAN.md`, and `TESTING.md` before coding.

## Pull Request Process
1. Branch using canonical naming conventions.
2. Keep task state synchronized between `PLAN.md` and project YAML.
3. Add a `BITACORA.md` entry for key changes.
4. Submit PR with linked tasks and test evidence.
